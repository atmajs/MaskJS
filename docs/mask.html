    ConditionUtil#

    Helper to work with conditional expressions

    condition(ternary, ?model)
    →result
    #
    - ternaryString
    - modelObject

    Data Model

    Ternary Operator is evaluated via ast parsing.
    All this expressions are valid:
          ('name=="me"',{name: 'me'}) -> true
          ('name=="me"?"yes"',{name: 'me'}) -> "yes"
          ('name=="me"? surname',{name: 'me', surname: 'you'}) -> 'you'
          ('name=="me" ? surname : "none"',{}) -> 'none'

    MaskDOM#

    This is just generic JSON Object with some contract structer to understand how mask.compile and mask.render works

    MaskDOM is a node nested tree, and there are 2 types of nodes: Literal and Tag

    Literal#
    { content: String | InterpolateFunction }
    Tag#
    {
      tagName: String,
      attr: {
            key: value // String | InterpolateFunction
      }
      nodes: // Node | [Node]
}
    InterpolateFunction(model, ?type = node, ?cntx, ?element, ?name)
    →Array
    #
    - modelObject

    Data Model used to render template

    - typeString

    node | attr

    returnArray

    When type is 'node', the array can also contain HTMLElements, otherwise only Strings

    This function will be created if parser meets interpolation parts, such as #{statement}, in Literal or in Tag.attr value.
